Task Dashboard MVP - Integration Test

Application Preview

Manual Test Instructions

  1. Clock Test: Verify the clock updates every second and shows current date
  2. Add Task Test:
    • Try adding a task with all fields filled
    • Try adding a task with empty name (should show error)
    • Try adding a task with no category (should show error)
  3. Task Management Test:
    • Add a task, then mark it as complete
    • Verify the progress bar updates
    • Delete the task and verify progress updates again
  4. Suggestion Test: Click "Ask Kiro for a Suggestion" multiple times and verify different suggestions appear
  5. Responsive Test: Resize the browser window to test mobile/desktop layouts

Automated Test Controls

Open the browser console and use these commands:

  • testDashboard.runAllTests() - Run all automated tests
  • testDashboard.addSampleData() - Add sample tasks for testing
  • testDashboard.clearAllTasks() - Clear all tasks
  • testDashboard.showStats() - Show current application statistics
